# Web3 Wallet Connection Basic

This project helps beginners learn how to connect a crypto wallet to a website and talk to the blockchain.

You will learn:
- How to connect MetaMask
- How to read wallet address
- How to check balance
- How to send a simple transaction
- How Web3 works in real projects

## Requirements
- Node.js installed
- MetaMask browser extension
- Basic JavaScript knowledge

## Setup

1. Clone the repo
git clone https://github.com/yourusername/web3-wallet-connection-basic.git

2. Install packages
npm install

3. Run the app
npm start

4. Open browser
http://localhost:3000

## Project Structure

- index.html = Main webpage
- app.js = Frontend logic
- web3.js = Blockchain connection
- wallet.js = Wallet functions
- config.js = Network settings
- server.js = Local server
- package.json = Project info
- utils.js = Helper functions
- logger.js = Logs
- .env.example = Environment example

## Features
- Connect wallet button
- Show wallet address
- Show wallet balance
- Send ETH transaction
- Log blockchain events

## Learning Goal
This project is made to help new developers understand how Web3 apps work in real life.
